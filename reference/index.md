# Package index

## Spotting functions and/or packages

Functions for identifying functions or packages in a single file or
collection of files.

- [`spot_funs()`](https://brshallo.github.io/funspotr/reference/spot_funs.md)
  : Spot Functions
- [`spot_pkgs()`](https://brshallo.github.io/funspotr/reference/spot_pkgs.md)
  : Spot Packages
- [`spot_tags()`](https://brshallo.github.io/funspotr/reference/spot_tags.md)
  : Spot Tags
- [`spot_funs_files()`](https://brshallo.github.io/funspotr/reference/spot_funs_files.md)
  [`spot_pkgs_files()`](https://brshallo.github.io/funspotr/reference/spot_funs_files.md)
  : Spot Packages or Functions in dataframe of Paths

## List files

Functions for listing the files in a repository or other location.

- [`list_files_github_repo()`](https://brshallo.github.io/funspotr/reference/list_files_github_repo.md)
  : List Files in Github Repo
- [`list_files_github_gists()`](https://brshallo.github.io/funspotr/reference/list_files_github_gists.md)
  : List Github Gists of User
- [`list_files_wd()`](https://brshallo.github.io/funspotr/reference/list_files_wd.md)
  : List Files in Working Directory
